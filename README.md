# Power BI TMDL Parser

Parse and analyze Power BI Tabular Model Definition Language (TMDL) files.

## Use cases

- **Dependency Tracking**: Analyze relationships and dependencies between tables, measures, and columns
- **SQL Querying**: Save the data to files or in database and query with your preferred language
- **AI Visualizations**: Generate insightful visualizations of your Power BI models using AI
